Brand information

Node.js is a cross-platform JavaScript runtime built on top of Google Chrome's V8 JavaScript engine that uses an event-driven, non-blocking I/O model that makes it lightweight and efficient.

Node.js API is built on Chrome V8 JavaScript engine together with AMD or CommonJS modules model

Node was once described as 'a impressive JavaScript snack that keeps going long after you're hungry' and for good reason! Node is a highly scalable platform that can be deployed in the cloud, on premise, or even within native mobile applications. Best of all, you can use any operating system to run Node apps because they are based on Google's open-source Chromium (Chrome) engine and have first rate, developer ergonomic support.
